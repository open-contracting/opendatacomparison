Title
SEAO - Published Contracts
Description
The dataset represents published contracts data by the Government of Quebec's e-procurement system, SEAO (Système électronique d’appel d’offres) in the OATS and from public organizations of Quebec, departments and agencies, agencies of the education system, network agencies of the health and social services as well as municipalities and other municipal agencies. However, note that publication in the OATS data municipal contracts is not mandatory since April 2011. Files will be added monthly. For the year, the files contain monthly data. For years prior, one file contains the data of every month.

Note: The September 15, 2013, changes were made to the rules on contracts. Data on additional costs and final contracts descriptions are provided in xml format in separate files, but at the same compressed file (.Zip) that notice. Specifications on the structure of files and fields are in the pdf document "Information on the distribution".
Title
Les contrats publiés
Description
L'ensemble de données ci-dessus concessionnaire contrats données publiées par le gouvernement de l'e-procurement le système québécois, SEAO (Système électronique d'appel d'Offres) dans la SEAO et de organizaciones publics de Québec, les ministères et organismes, les agences du système de l'éducation, réseau d'agences la santé et des services sociaux ainsi que des municipalités et d'autres organismes locaux. Toutefois, notez que la publication dans les contrats de données de AVOINE locaux n'est pas obligatoire depuis Avril 2011. Les fichiers seront ajoutés chaque mois. Pour l'exercice, les fichiers mensuels de données contiennent. Pour les années antérieures, un fichier contient les données de tous les mois.

Remarque: Les 15 Septembre 2013, étaient les modifications apportées aux règles sur les marchés. Les données sur les coûts supplémentaires et les contrats finaux descriptions sont fournies au format XML dans des fichiers séparés, mais dans le même fichier compressé (. Zip) C'est de préavis. Spécifications relatives à la structure des fichiers et les champs sont dans le document pdf "information sur la distribution".
