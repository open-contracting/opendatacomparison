Title
OpenTED - Contract Awards
Description
The dataset is available from Open Tenders Electronic Daily (OpenTED). OpenTED releases updated data extracts from the European procurement database, Tenders Electronic Daily (TED). TED data is shared by the EU Publications Office in a XML format. While that format is well-suited to automated processing, it doesn't easily lend itself to analysis by journalists. Therefore, OpenTED provides the data as CSV (comma-separated values) files, which can be read and analysed by common spreadsheet programs like Excel or LibreOffice. As TED is a very large database, extracts are divided into slices by country and year. This should make the data small enough to be handled in desktop applications.

Contracts extracts contain only the subset of TED activities which relates to contract awards. This data includes the supplier names and contract amounts for contracts which have been awarded to a specific company. Document extracts contain all activities published in TED, including tender announcements, metadata on contract awards etc. Use this data if you want to get an overview of the activities in European procurement. Note that the data dictionary is currently a work in progress.

Data is available by year (2012-2014) and country from the European Union. The dataset has the most fields (a total of 99) out of all datasets analyzed thus far.

The Publications Office of the European Union claims copyright on the TED data. Resources made available on this website are subject to those terms, but no additional copyright claim is made by the authors of this site.
