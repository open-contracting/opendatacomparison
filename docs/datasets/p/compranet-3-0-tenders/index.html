Title
CompraNet - CompraNet 3.0 Tenders
Description
The dataset covers derivative contract data and procurement procedures reported in the Compranet 3.0, between 2002 and 2011. Reports are available in Excel format. Data is available from downloads on Competitive Bidding and Exceptions to Competitive Bidding. This is a particularly interesting unique feature encountered in tender datasets.

The dataset is published by Agencies and Entities of the Federal Government and the State Governments.

The description of fields is available in free text on the website links but does not include all fields in the dataset.

Title
CompraNet - Contratos Compranet 3.0
Description
El conjunto de datos contiene datos de contratos de derivados y los procedimientos de contratación publicados en la Compranet 3.0, entre 2002 y 2011. Los informes están disponibles en formato Excel. Los datos están disponibles a partir de las descargas en la Licitación Pública y Excepciones a la Licitación Pública. Esta es una característica única particularmente interesante encontrado en los conjuntos de datos de contratos.

El conjunto de datos es una publicación de las Dependencias y Entidades del Gobierno Federal y los gobiernos estatales.

La descripción de los campos está disponible en texto libre en los enlaces de sitios web, pero no incluye todos los campos del conjunto de datos.
