Description
Data is available on contracts published in the electronic tendering system (SEAO) and from public organizations of Quebec, departments and agencies, agencies of network education, network organizations of health and social as well as municipalities and other municipal agencies.

However, note that publication in the SEAO data municipal contracts is not mandatory since April 2011. Files will be added monthly.

For the year, the files contain monthly data. For years prior, one file contains the data of every month.

IMPORTANT NOTE : The September 15, 2013, changes were made to the rules on contracts. Data on additional costs and final contracts descriptions are provided in xml format in separate files , but at the same compressed file (.Zip) that notice. Specifications on the structure of files and fields are in the pdf document "Information on the distribution ."
