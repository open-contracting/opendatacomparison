Visualization of datamap

CSV

The datamap applies to the Tender Data download available in the Tender Data dataset published by Tender Monitor. Originally, the dataset included 15 bidders with 4 attributable fields for each bidder (60 fields total). We did not include all of the individual bidders in different fields pertaining to bidder information but we consolidated them: e.g., bidder_x_name; bidder_x_id; bidder_x_lowest_bid; bidder_x_black_or_white. Data is available in both Georgian and English in a single CSV file. There is no data dictionary available so we had to infer the descriptions of the fields.
