The datamap applies to the Contract Awards (2013) download in the Contract Awards dataset published by OpenTED. This datamap has the most fields of datamaps analyzed thus far. It includes detailed information on award data pertinent to European Union tender information. There is no complete data dictionary currently available so we had to infer the descriptions of the fields. UPDATE: Since the analysis of this dataset, the OpenTed site has renamed all the fields in the downloadable data, so this datamap does not apply to the currently available datasets.
